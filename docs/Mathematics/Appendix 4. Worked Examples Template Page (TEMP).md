# Appendix 4. Worked Examples Template Page (TEMP) {-}

| **Worked Example I**                                              |
| ----------------------------------------------------------------- |
| Find $u_{10}$ for an arithmetic sequence where $u_1=3$ and $d=5$. |
| We use the $n^{th}$ term formula:                                 |
| $$u_n = u_1 + (n-1)d$$                                            |
| $$u_{10} = 3 + (10-1) \cdot 5 = 3 + 45 = 48$$                     |
