::: example
**Worked example: Arithmetic sequence**

Given $u_1=3$ and $d=5$, find $u_{10}$.

$u_{10}=u_1+(n-1)d=3+9\cdot5=48$
:::

::: exercise I
Find $u_{15}$ for $u_1=5, d=3$.
:::

::: exercise II

:::
